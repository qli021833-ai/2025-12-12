常见问题解答

温馨提示：点击右侧 符号即可打开答案，再次点击即可折叠。

1. 商标注册需要准备哪些材料？

基础材料包括：
• 申请人身份证明（企业提供营业执照，个人提供身份证）
• 商标标样（清晰的LOGO图片，部分国家标准字体申请无需提供logo）
• 商品/服务项目清单（需符合对应国家的）
• 委托书/申请书
不同国家可能有额外要求：• 美国：需提供使用证据或意向使用声明 • 欧盟：无额外材料，基础材料即可 • 马德里体系：需国内商标申请/注册证明
我们会根据具体国家提供详细材料清单，并协助整理审核。

2. 商标注册周期是多久？

不同国家/地区注册周期不同：
• 美国：8-12个月
• 欧盟：4-6个月
• 英国：4-6个月
• 日本：8-12月
注：以上为顺利下证的周期，若遇到审查意见/异议，周期会相应延长。

3. 商标注册成功率是多少？如何提高成功率？

成功率主要因素是商标名和使用证据，提高成功率的方法：
1. 注册前做精准的商标检索，避开相同/近似商标；
2. 商标标样避免使用通用词、描述性词汇；
3. 选择专业代理机构，提前规避驳回风险；
4. 若收到审查意见，及时专业答辩。

4. 亚马逊品牌备案需要什么条件？备案失败怎么办？

备案条件：
• 拥有已注册的商标（R标）或受理中的商标（TM标，部分站点支持）；
• 商标标样需与产品包装/Listing一致；
• 拥有对应的亚马逊店铺。
备案失败解决方法：
1. 检查商标信息与店铺信息是否一致；
2. 补充商标使用证据（如产品图片、包装图）；
3. 联系亚马逊品牌支持团队申诉；
4. 委托专业代理机构协助处理（可提供协助备案操作服务）。

5. 商标注册后需要注意什么？有哪些维护义务？

注意事项及维护义务：
1. 及时使用商标：避免“撤三”风险（如中国3年未使用可能被撤销）；
2. 按时续展：商标有效期通常为10年，需提前6-12个月续展；
3. 美国商标需做“五年宣誓”：注册后5-6年需提交使用证据；
4. 监控商标状态：及时处理异议、无效宣告等风险；
5. 变更信息需备案：如申请人地址/名称变更，需及时办理商标变更。

6. 遇到TRO投诉/侵权指控该怎么办？

应对步骤：
1. 立即停止销售被控侵权产品，避免损失扩大；
2. 收集证据：确认是否真的侵权（对比商标/专利信息）；
3. 联系专业知识产权律师/代理机构）评估风险；
4. 选择应对方式：
• 若侵权成立：协商和解、支付赔偿金；
• 若不侵权：提交不侵权证据应诉；
5. 后续预防：完善知识产权布局，避免再次侵权。

7. 专利申请与商标注册有什么区别？

核心区别：
• 保护对象不同：商标保护“品牌标识”（如LOGO、名称）；专利保护“技术/设计方案”（如产品外观、发明技术）；
• 保护期限不同：商标可无限续展（每10年续展一次）；专利有固定期限（外观专利15年，发明专利20年）；
• 申请条件不同：商标需具有“显著性”；专利需具有“新颖性、创造性、实用性”；
• 作用不同：商标用于区分商品/服务来源；专利用于保护技术创新，防止他人抄袭。
